# kINGROON TITAN EXTRUDER COOLER
![titan Extruder coolor](./kingroon_titan_cooling/image/Image01.png)
## 필요한 부품 리스트
* M4
    * 6mm 3개(본체결합용)
* M3 
    * 20mm 5개 (4010,5015 팬 고정용)
    * 너트 2개 (5015팬 고정용)
    * 35mm 1개 (4010<>5015 부속 연결)
* 자석
    3파이 3MM 4개 (덕트고정용)

### 재질
* PLA 가능

### 설계 기준 히팅 시스템
* E3d V5, V6
* 높이가 30mm인 노즐목 사용 기준
